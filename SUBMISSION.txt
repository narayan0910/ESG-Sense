Instructions:
1. Run `npm install` in both `server/` and `client/` directories.
2. Run `node seed.js` in `server/` to initialize the database.
3. Start backend: `node index.js` inside `server/`.
4. Start frontend: `npm run dev` inside `client/`.
5. Access via http://localhost:5173.
